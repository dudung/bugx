two slit interference function resultan

Mon, 28 Mar 2022

Interferensi dua celah [1].

electric field#

Medan listrik dari masing-masing sumber dapat dinyatakan sebagai

\begin{equation}\label{eqn:two-slit-interference-electric-field} E_j = E_0 \sin (kx_j - \omega t + \phi), \ \ \ \ j = 1, 2, \end{equation}

dengan $k$ bilangan gelombang, $x$ posisi suatu titik, $\omega$ frekuensi angular, $t$ waktu, dan $\phi$ adalah fasa.
