github repo site markdown

Sun, 12 Jun 2022

Tulisan ini mungkin dapat disebut sebagai tutorial untuk membuat situs web dari suatu repositori di GitHub dan merupakan kelanjutan dari tulisan lain [1]. Terkait dengan real repo.

prerequisite#

Pembaca tulisan ini diasumsikan memiliki akun GitHub username dan telah membuat situs web suatu repositori repository dan mengaktifkannya dengan alamat seperti https://username.github.io/repository, seperti contoh yang digunakan di sini https://dudung.github.io/simple-website-md.
